What is SCEAP 2.0?

SCEAP = Smart Cable Engineering Automation Platform

This is an intelligent web application designed to automate cable engineering workflows in power plants and buildings. Instead of manually calculating cable sizes in Excel spreadsheets, engineers can now:

  • Upload a list of electrical feeders (Excel file)
  • Let the system automatically discover cable paths from equipment to transformers
  • Calculate optimal cable sizes based on current, voltage drop, and short-circuit protection
  • Export results in Excel or PDF format
  • Make data-driven decisions with visual analysis

Why Was This Built?

Problem Solved: Manual cable sizing calculations are time-consuming, error-prone, and hard to document.

Solution Provided: Automated, accurate, repeatable, and exportable cable sizing calculations.

Impact: Engineers save days of work per project and reduce calculation errors.

Why Is the Project Organized This Way?

The project is split into two completely separate applications:
  • sceap-frontend/ - The user interface (React application)
  • sceap-backend/ - The server and business logic (.NET application)

This separation of concerns provides scalability, maintainability, and independent deployment. The frontend can be updated without touching the backend and vice versa.
